# 🎵 Music Playlist using Doubly Circular Linked List

## 📌 Project Overview
This project is part of my **100 Days DSA Challenge**.

It implements a **Music Playlist System** using a **Doubly Circular Linked List (DCLL)** and provides an **interactive web-based UI** built with HTML, CSS, and JavaScript.

---

## 🚀 Features

- ➕ Add songs to playlist  
- ❌ Delete current song  
- ▶️ Play next song  
- ⏮ Play previous song  
- 🎧 Highlight currently playing song  
- 🔁 Circular playlist traversal  

---

## 🧠 Data Structure Used

### Doubly Circular Linked List

Each node contains:
- `song` (name)
- `next` pointer
- `prev` pointer  

### Why DCLL?

- Allows **bidirectional traversal**
- Enables **O(1) next/previous navigation**
- Supports **continuous looping (circular behavior)**
